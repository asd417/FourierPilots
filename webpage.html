Abstract

Previous research projects demonstrate that signal processing techniques can be utilized in neural networks to improve the performance of the overall agent. This research project builds up on this idea to improve the performance of a behavior-cloning xpilot agent.

Method

We apply a Fast Fourier Transform (FFT) and Discrete Cosine Transform (DCT) to compress high-dimensional observations from the xpilot environment before passing them to a Neural Network (NN) controller trained via Behavior Cloning on data generated by a rule-based agent. For comparison, we also train a baseline NN controller without observation compression. We evaluate both agents on accuracy and validation loss.
